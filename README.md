# tiktok-auth

A lightweight TikTok OAuth 2.0 authentication integration that lets users sign in with their TikTok account.

## Overview

**tiktok-auth** handles the TikTok OAuth flow — redirecting users to TikTok's authorization page, receiving the callback, and exchanging the code for an access token. It is hosted as a static site on GitHub Pages.

**Live site:** https://nuttapong14.github.io/index.html

## Features

- TikTok OAuth 2.0 authorization flow
- Secure token exchange via callback handling
- Minimal footprint — no backend server required for the auth redirect
- Privacy-first: only collects data necessary to authenticate
